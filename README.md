# Jenny-Killer Pack

Skills de redacción científica y profesional pa’ Perplexity/Claude/Gemini. Cero hype, cero citas fake.

## Skills incluidos

| Skill | Uso | Comandos |
| --- | --- | --- |
| **jenny-killer** | Papers Q1, tesis doctoral, stats duros | `/paper /methods /results /abstract /reviewer2` |
| **redaccion-pro** | Ensayos, tesis, LinkedIn, reports | `/mejorar /estructura /tono /pulir /tesis` |
| **seo-killer** | Blogs, keywords, meta, FAQ schema | `/blog /keyword /meta /faq /outline` |

## Instalación Perplexity

1. Ve a Releases → Baja `jenny-killer.zip` o el que quieras
2. Perplexity → Settings → Skills → Upload Skill → Sube el `.zip`
3. Nuevo chat → Escribe `/paper` o `/mejorar` y jala
